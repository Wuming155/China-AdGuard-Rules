### 🔗 来源说明

1. **SM-Ad-FuckU-hosts**: 针对中国安卓 APP 优化的去广告 Hosts，主要拦截国内主流 APP 的开屏广告及穿山甲、腾讯、百度、快手等广告联盟。
2. **AdRules**: 针对中国地区优化的广告拦截列表，提供 AdBlock、DNS 及多种代理工具格式，涵盖全面。
3. **AWAvenue-Ads-Rule**: 强大的类 Adblock 规则库（秋风广告规则），作为众多规则的上游，以极小体积实现高命中率的广告拦截。
4. **DD-AD**: 综合性广告过滤规则，特别针对番茄小说、七猫小说等应用进行了私有规则优化。
5. **banad**: 轻量级广告拦截规则（大萌主），专注于去除色情悬浮广告，兼容大部分浏览器和 AdGuard 扩展。
6. **HG**: 稳定的 AdGuardHome 规则（海哥拦截规则），更新频率较慢但规则相对稳定，适用于网络层面拦截。
7. **qq5460168/666**: 强力去广告规则，适配多种工具（如 Clash、SingBox、Shadowrocket），由酷安社区成员维护，拦截覆盖广。
